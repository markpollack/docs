## Overview

Java SDK for the Agent Communication Protocol (ACP) — a standard protocol for agent-to-agent and agent-to-IDE communication. Build agents, consume agents, and integrate with IDEs using a consistent API.

## Tutorial Structure

The tutorial covers four parts, from first contact to IDE integration:

| Part | Topic | Modules |
|------|-------|---------|
| 1 | Client Basics | Protocol basics, sessions, prompts, streaming, agent requests, permissions, error handling |
| 2 | Building Agents | Echo agent, handlers, updates, requests, testing, capability negotiation, MCP servers |
| 3 | Advanced | Async client, async agent |
| 4 | IDE Integration | Zed, JetBrains, VS Code integration |

## Role in the Lab

ACP provides the communication backbone for multi-agent architectures. As the lab evolves toward agent-to-agent workflows (see [Spring AI A2A](/projects/spring-ai-a2a)), ACP standardizes how agents discover, connect, and delegate to each other.
