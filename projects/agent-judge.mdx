## Overview

Agent Judge provides a principled way to evaluate AI agent work products. Instead of relying solely on LLM-as-judge (expensive, non-deterministic), it implements a **four-tier cascaded jury** that filters through deterministic checks first.

## The Four Tiers

| Tier | Name | What It Does | Cost |
|------|------|-------------|------|
| **T0** | Deterministic | Regex, file existence, compilation checks | Free |
| **T1** | Command | Run shell commands, check exit codes | Minimal |
| **T2** | Golden Test | Compare against known-good reference outputs | Minimal |
| **T3** | LLM Assessment | Semantic evaluation by language model | $$$ |

Each tier acts as a filter — only work products that pass cheaper tiers advance to expensive LLM evaluation. This reduces cost by 60-80% while maintaining evaluation quality.
