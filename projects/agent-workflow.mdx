## Overview

Agent Workflow is the execution engine at the heart of the lab's agent stack. It provides two approaches for building AI agents with Spring AI:

- **AgentLoopAdvisor** — Drop-in Spring AI advisor that adds autonomous agent behavior to any chat client
- **Explicit loop patterns** — Full control over the agent execution cycle for SDLC-focused agents

## Why It Matters

Most agent frameworks focus on prompt engineering. Agent Workflow focuses on **execution infrastructure** — the loop control, tool orchestration, and state management that determine whether an agent completes reliably or spirals.

This is the "structured execution" half of our thesis: *knowledge + structured execution > model*.
