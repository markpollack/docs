Agento Studio is a methodology and toolkit for building, testing, and improving AI agent knowledge bases. Built on the **Forge methodology** — a six-phase system that separates discovery from execution and treats evaluation as first-class.

## Quick Start

```bash
git clone https://github.com/markpollack/agento-studio.git
cd agento-studio
claude
# then: /forge-kb ~/path/to/your/docs
```

From another project, use `--add-dir`:

```bash
claude --add-dir ~/agento-studio
```

## Slash Commands

| Command | What it does |
|---------|-------------|
| `/forge-project` | Scaffold a software project (Vision + Design + Roadmap) |
| `/forge-research` | Bootstrap a research project |
| `/forge-kb` | Structure a corpus for JIT context Q&A |
| `/forge-eval-agent` | Set up agent evaluation with judges |
| `/forge-steward` | Bootstrap ongoing project stewardship |
| `/forge-research-kb` | Bootstrap a federated research-partner KB |
| `/plan-to-roadmap` | Convert a plan into a tracked roadmap |
| `/collect-status` | Produce a timestamped status report |

## What's Inside

**8 slash commands** — executable from Claude Code, not documentation to read.

**13 concept docs** — Knowledge base architecture, discovery loops, judges, steward agents, research methodology.

**11 templates** — Vision, design, roadmap, learnings, paper tracker, provenance — fill-in-the-blank for every phase output.

**4 project variants** — Eval-agent, project, research, steward. Same six phases, different feedback loops.
