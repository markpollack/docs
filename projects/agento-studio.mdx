Agento Studio is an interactive environment for creating autonomous agents. Design agent workflows visually, wire up tools and skills, configure judge rubrics, and deploy — all without leaving the studio.

## What It Does

- **Visual agent composition** — drag-and-drop steps, tools, skills, and quality gates into agent workflows
- **Live preview** — test agent behavior against sample tasks before deploying
- **Skills integration** — browse and attach SkillsJars from Maven Central or local collections
- **Judge configuration** — define tiered jury rubrics with deterministic and LLM judges
- **One-click deploy** — export as a Spring Boot application or AgentJar artifact

## Built On

Agento Studio builds on the AgentWorks stack — [Agent Workflow](/projects/agent-workflow) for pipeline composition, [Agent Judge](/projects/agent-judge) for evaluation, and [Agent Skills](/projects/agent-skills) for domain knowledge.
