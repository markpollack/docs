The lab produces a focused set of projects that work together as an agent engineering stack. Each project is open-source and built on [Spring AI](https://spring.io/projects/spring-ai).

## The Stack

```
┌─────────────────────────────────────────┐
│           Loopy (CLI Agent)             │  ← Interactive coding agent
├─────────────────────────────────────────┤
│  Agent Client  │  Agent Workflow        │  ← Execution layer
├────────────────┼────────────────────────┤
│  Claude SDK    │  ACP Java SDK          │  ← SDKs & Protocols
├─────────────────────────────────────────┤
│  Agent Judge   │  Agent Bench           │  ← Evaluation layer
├─────────────────────────────────────────┤
│          Agent Journal                  │  ← Observability layer
└─────────────────────────────────────────┘
```
